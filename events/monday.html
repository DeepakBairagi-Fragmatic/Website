About This Workshop

Dive deep into the world of user interface design in this comprehensive 2-hour bootcamp. You'll learn the essential principles that make great user interfaces, from color theory to typography, spacing, and layout composition.

We'll be using industry-standard tools like Figma for design and Tailwind CSS for rapid prototyping. By the end of this session, you'll have created your own portfolio-worthy landing page design.

What You'll Learn:

  • Design system fundamentals and component architecture
  • Advanced Figma techniques and shortcuts
  • Tailwind CSS utility-first approach
  • Responsive design best practices
  • Designing for accessibility and inclusivity
